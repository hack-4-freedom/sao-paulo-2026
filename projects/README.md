# Projetos

Esta pasta contém resumos e links de todos os projetos construídos durante o **Hack4Freedom São Paulo 2026**.

Cada projeto foi desenvolvido por uma equipe de participantes como parte do hackathon de duas semanas e é:
- Open-source por padrão
- Construído em repositórios públicos no GitHub
- Focado em resolver problemas reais locais usando Bitcoin e tecnologia de liberdade

Este diretório serve tanto como registro do trabalho realizado durante o Hack4Freedom São Paulo quanto como ponto de partida para o desenvolvimento contínuo após o hackathon.

Consulte [`how-to-add-your-project.md`](how-to-add-your-project.md) para instruções passo a passo de como enviar a documentação do seu projeto via pull request.
