# Roadmap

## Versão 0.3 — Fundação (atual)

- [x] Cadastro e login (email/senha)
- [x] Auto-provisioning (profile + wallet + friend_code)
- [x] Trilhas e lições com story frames e quizzes
- [x] Sistema de XP e níveis
- [x] Missões diárias
- [x] Badges (35+)
- [x] Streak (sequência diária)
- [x] Carteira Bitcoin (Breez SDK Spark)
- [x] Perfil com username, bio, links, banner
